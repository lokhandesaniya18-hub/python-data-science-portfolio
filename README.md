# Project 1: iPhone Sales Analysis

**Tools Used:** Python, Pandas, Jupyter Notebook  
**Dataset:** iPhone Sales Data - iphone_sales.csv

### **Project Objective**
Analyze iPhone sales data to find top selling models, highest revenue cities, and revenue trends.

### **What I Did**
1.  **Data Loading:** Loaded CSV data using Pandas
2.  **Data Cleaning:** Handled missing values and created new column `TotalAmount = Price * Quantity`
3.  **Analysis:** 
    - Found Top Selling iPhone Model using `groupby`
    - Found City with Highest Revenue
    - Filtered Premium iPhones > 1 Lakh
4.  **Output:** Saved final analysis report as `iphone_sales_report.csv`

### **Key Skills Used**
Data Cleaning, Data Analysis, Pandas `groupby`, Filtering, CSV Export

### **Key Insight**
This project helped me understand how to extract business insights from raw sales data using Python.
